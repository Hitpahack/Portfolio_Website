Project

The current data management system for Falca team's outlets is causing inefficiencies in data gathering and analysis, hindering the ability to make informed strategic and business decisions. This poses a significant challenge for UX/UI designers to create a user-friendly interface that can effectively streamline data collection and analysis processes for improved decision-making.

Challenge

Each user needed custom data views. It was crucial to avoid overwhelming users with excessive information. Designing within Figma while considering Power BI's layout constraints was a primary challenge. Ensuring consistent functionality across devices and seamless real-time data integration were also essential.

DIRECTOR DASHBOARD

The director dashboard is meticulously crafted with user-centric design and a PowerBI layout to ensure seamless and intuitive navigation. Key functions and features are thoughtfully arranged for easy accessibility, boasting a logical layout that effectively minimizes the learning curve.
